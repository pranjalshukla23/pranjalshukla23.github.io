Project Information:

In this section we will explain the steps that were followed for developing this project

  1. Preparing the dataset:

    2. The dataset used in this model comprise of following columns:

    • outlook : Weather on that day
    • temp : tempreature on that day
    • humidity : humidity on that day
    • windy : Informs if the day is windy or not
    • play : Informs if the match will be played or not

  2. Preprocessing the data:

    3. We are now selecting the columns that will be given as input to our model
    The input columns are:
    • outlook
    • temp
    • humidity
    • windy

    The Output column is:
    • Play

  3. Data Visualization
    4. In this step, we explain the data through graphs mainly barplot.
    we will plot the values of 'play' against each feature variables

  4. Spliting the data into training and testing parts:

    5. In this step we split the data using train_test_split function in sci-kit learn library of python,the training data will be 80% of the whole dataset and testing data will be 20% of whole dataset

  5. Normalizing the data:

    6. In this step we normalize the data using StandardScaler function in sci-kit learn library of python

  6. Training the data:

    7. In this step we will use Decision tree algorithm to make the prediction

  7. Calculating the accuracy:

    8. In this step we will calcualte the accuracy of our model

  8. Testing the model:

    9. In this step we will provide our model a sample input and get the predicted score
