Project Information:

In this section we will explain the steps that were followed for developing this project

  1. Preparing the dataset:

    2. The dataset used in this model comprise of following columns:

    • mid : Unique index for each match
    • date : date on which match is played
    • bat_team : name of batting team
    • bowl_team : name of bowling team
    • batsman : batsman name
    • Wickets : number of wickets taken till now
    • bowler : bowler name
    • Runs : The number of runs scored till now in the match
    • Overs : The number of overs bowled till now in the match
    • Striker : The runs scored by the striker till now
    • Non-Sriker : The runs scored by the non-striker till now

  2. Preprocessing the data:

    3. We are now selecting the columns that will be given as input to our model
    The input columns are:
    • Runs
    • Overs
    • Wickets
    • Striker
    • Non-Striker

    The Output column is:
    • Total

  3. Spliting the data into training and testing parts:

    4. In this step we split the data using train_test_split function in sci-kit learn library of python,the training data will be 80% of the whole dataset and testing data will be 20% of whole dataset

  4. Normalizing the data:

    5. In this step we normalize the data using StandardScaler function in sci-kit learn library of python

  5. Training the data:

    6. In this step we will use Random forest algorithm to make the prediction
